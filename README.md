# 나의 할 일 관리 앱

## 1. 앱 소개
이 앱은 내가 해야할 일을 적고 그 일을 완수했으면 완료를 눌러 내가 무슨 일을 하고 어떤 일을 완료 했는지 보는 앱입니다.

## 2. 사용한 AI
Gemini

## 3. 처음에 입력한 프롬프트
HTML, CSS, JavaScript를 하나의 index.html 파일에 넣어서 할 일 관리 앱을 만들어줘. 할 일을 입력하고 추가할 수 있고, 완료 표기와 삭제 기능이 있었으면 좋겠어. 초보자도 이해할 수 있게 코드를 너후 복잡하게 만들기 말아줘.

## 4. 수정 요청한 내용
처음에 배경색을 흰색으로 했다가 밝은 초록색으로 바꿔달라함

## 5. 어려웠던 점 또는 배운 점
코드 하나하나를 내가 쓰고 실해해야만 하는 건 줄 알았는데 이렇게 쉽고 간단하게 할 수 있는 것을 배웠다.
